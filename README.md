# FastAPI Backend Service

Backend-часть сервиса на FastAPI с Docker-контейнеризацией.

## Технологии

- Python 3.11
- FastAPI
- Uvicorn (ASGI-сервер)
- Docker + Docker Compose
- Pydantic (валидация данных)

## Требования

- Docker 20.10.0+
- Docker Compose 2.0.0+
- (Опционально) Python 3.11+ для локальной разработки
